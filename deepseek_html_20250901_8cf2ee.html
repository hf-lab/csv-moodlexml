CSV to Moodle XML Converter

Instructions

This tool converts CSV (Comma-Separated Values) files into Moodle XML format for importing questions into Moodle.

  1. Prepare your CSV file with the following columns: type, name, question, generalfeedback, answer1, answer2, ...
  2. For multiple choice questions, mark correct answers with a preceding * asterisk
  3. Either paste your CSV data or upload a CSV file
  4. Click the "Convert to XML" button to generate Moodle XML
  5. Download the generated XML file and import it into Moodle
